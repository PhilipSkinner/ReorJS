<code> use ReorJS;

my $reorjs = ReorJS->new(); $reorjs->setHost('localhost'); $reorjs->setPort('9999'); $reorjs->setKey('temporary');

my $applications = $reorjs->listApplications(); </code>

Methods

ReorJS->new()

Creates a new ReorJS API connector object.

Can receive optional hash of parameters. Supported parameters are:

* host * port * key

$reorjs->setHost($hostname)

Sets the reorjsd hostname to be used for requests

$reorjs->setPort($port)

Sets the port to be used for communication

$reorjs->setKey($key)

Sets the access key to be used for authentication

$reorjs->detailTask($id)

Returns details for a particular task in the system

$reorjs->createTask($params)

Creates a task in the system

Receives a hash of parameters. Supported parameters are:

* application * dataset * result

$reorjs->listTasks()

Lists all of the tasks currently in the system

$reorjs->createDataset($params)

Creates a new dataset (or source) in the system

Receives a hash of parameters. Supported parameters are:

* name * source_type * source_hostname * source_port * source_name * source_table * source_username * source_password

$reorjs->modifyDataset($id, $params)

Modifies an existing dataset

Receives a hash of parameters. Supported parameters are:

* name * source_type * source_hostname * source_port * source_name * source_table * source_username * source_password

$reorjs->deleteDataset($id)

Deletes a dataset

$reorjs->detailDataset($id)

Returns the details for a dataset

$reorjs->listDatasets($params)

Returns a list of datasets in the system

$reorjs->createApplication($params)

Creates a new application

Receives a hash of parameters. Supported parameters are:

* name * program

$reorjs->modifyApplication($id, $params)

Modifies an application

Receives a hash of parameters. Supported parameters are:

* name * program

$reorjs->deleteApplication($id)

Deletes an application

$reorjs->listApplications($params)

Lists all of the applications
